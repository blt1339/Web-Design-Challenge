Comparison: Latitude vs. X

The purpose of this project was to analye how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset of over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs latitude. Factores we looked at included: temperature, cloudiness, wind speed, and humidity. This site proides the source data and visualizations created as part of the analysis, as well as explainations and descriptions of any trends and correlations witnessed.
